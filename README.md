# Individual Project

This project is intended to reproduce tables and figures generated by the New York Times related to COVID-19 as of January 17, 2021. 

## Files
- `270_individual_project.Rmd`: code used to reproduce tables and figures from the New York Times
- `270_individual_project.pdf`: knitted PDF from Rmarkdown (code at end of document)
-  `all-states-history.csv`: COVID-19 hospitalizations by state, downloaded on January 19, 2021, from [The COVID Tracking Project](https://covidtracking.com/data) via the CSV hyperlink
- `us-counties.csv`: COVID-19 cases and deaths by county, downloaded on January 19, 2021, from the [New York Times](https://github.com/nytimes/covid-19-data) GitHub repository
- `us-states.csv`: COVID-19 cases and deaths by state downloaded on January 19, 2021, from the New York Times GitHub repository

## Computing Information
**Operating System**: macOS Big Sur (version 11.0.1)

**Software**: R (version 4.0.3)

**Packages**
- `knitr` (version 1.30): knitting RMarkdown into PDF/HTML format
- `tidyverse` (version 1.3.0): data wrangling and visualization
- `zoo` (version 1.8-8): analyzing time-series data 
- `kableExtra` (version 1.3.1): displaying data tables
- `usmap` (version 1.5.1): plotting US maps, contains US population data *package is archived, install using the code below* 
  `install.packages('https://cran.r-project.org/src/contrib/Archive/usmap/usmap_0.5.1.tar.gz', repos=NULL, type="source")`
